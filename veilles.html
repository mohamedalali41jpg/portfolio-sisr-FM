Veille Technologique

Sujet : Les vulnérabilités critiques et la sécurisation des infrastructures (2025-2026).

Mars 2026 : Faille Shadow Admin

Analyse de la vulnérabilité CVE-2026-1422 sur Active Directory. Elle permet une attaque DCSync pour usurper l'identité d'un contrôleur de domaine et extraire les mots de passe.

Source : Microsoft MSRC

Février 2026 : Contournement MFA

Étude des techniques de "Session Hijacking" permettant de voler des cookies de session après une authentification Multi-Facteurs (MFA). Remédiation via les clés FIDO2.

Source : CERT-FR

Janvier 2026 : Vulnérabilité Cluster DoS

Identification d'une faille dans le service de clustering de Windows Server 2022 provoquant des arrêts de service critiques (BSOD) sur les serveurs de fichiers en haute disponibilité.

Source : iTPro.fr

Décembre 2025 : Attaques par Relais NTLM

Analyse des risques d'élévation de privilèges vers "Domain Admin". Utilisation de la signature SMB et de Kerberos pour protéger l'infrastructure Active Directory.

Source : ANSSI

Novembre 2025 : Failles RCE sur VPN SSL

Alerte sur les passerelles VPN SSL permettant l'exécution de code à distance (RCE) sans authentification. Nécessité de mettre à jour les Firmwares des pare-feu de bordure.

Source : FortiGuard Labs

Octobre 2025 : Corruption du flux WSUS

Exploitation du service de mise à jour WSUS pour injecter des scripts malveillants sur les postes clients. Passage obligatoire au protocole HTTPS/TLS pour sécuriser les flux.

Source : Microsoft Security
