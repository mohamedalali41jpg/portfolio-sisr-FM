Mes projets

Voici un aperçu de quelques projets réalisés durant ma formation en BTS SIO option SISR. Ces travaux m’ont permis de mettre en pratique des compétences en réseaux, systèmes, virtualisation et sécurité.

Installation de Wampserver et configuration de FileZilla

Dans le cadre d’un travail pratique, j’ai installé Wampserver sur un serveur Windows afin d’héberger des applications web (Apache, PHP et MySQL). Après avoir vérifié les modules requis (Visual C++ Packages) et assuré le bon fonctionnement via l’outil Check_vredist, j’ai configuré la plateforme pour accéder à phpMyAdmin depuis un poste client.

La seconde partie consistait à mettre en place un serveur FTP avec FileZilla Server : création d’un utilisateur (dev_gsb), définition des droits (lecture, écriture, suppression) et choix d’un répertoire partagé. Un port range personnalisé (42000‑42300) a été défini et ouvert dans le pare‑feu Windows. Une règle entrante et sortante a été ajoutée pour les ports 21, 80, 443 et 990 afin de permettre les connexions FTP/FTPS.

Une fois la configuration terminée, la connexion avec FileZilla Client a permis de transférer des fichiers entre le poste client et le serveur via l’adresse IP locale. Ce projet a également inclus le déploiement d’un site web et la modification d’un fichier de configuration PHP afin de corriger un problème d’authentification.

Compétences : Wampserver, Windows Server, FileZilla Server, phpMyAdmin, pare‑feu Windows

Mise en place de services réseau (DHCP, DNS, Active Directory)

Déploiement et configuration de services réseau essentiels pour une infrastructure d’entreprise : serveur DHCP pour l’attribution d’adresses IP, serveur DNS pour la résolution de noms, et annuaire Active Directory (Windows Server 2019) pour la gestion des utilisateurs et des postes. Mise en place du routage et du NAT/PAT pour l’accès Internet, ainsi que des filtrages ACL sur les commutateurs Cisco.

Compétences : DHCP, DNS, Active Directory, routage, NAT/PAT, ACL Cisco

Administration de postes Windows & Linux

Gestion et maintenance de parcs informatiques sous Windows 10/11 et distributions Linux. Création de comptes utilisateurs, définition des droits d’accès (NTFS, FAT), configuration des stratégies de groupe (GPO) et déploiement d’outils de supervision. Mise en place de scripts d’automatisation pour la sauvegarde et la mise à jour des postes.

Compétences : Windows Server, Linux (Ubuntu/Debian), administration système, gestion des droits, scripts Bash/PowerShell

Virtualisation et conteneurisation

Utilisation d’outils de virtualisation (Hyper‑V, Proxmox) et de conteneurisation (Docker) pour créer des environnements isolés. Mise en place d’un cluster de machines virtuelles pour simuler une infrastructure complète et déploiement d’une application multi‑services avec Docker Compose.

Compétences : Hyper‑V, Proxmox, Docker, réseaux virtuels, orchestration de conteneurs
