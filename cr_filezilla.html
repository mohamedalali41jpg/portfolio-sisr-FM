Projet FileZilla & Wamp

L'objectif de ce projet était de mettre en place une plateforme complète d'hébergement web et de partage de fichiers sur un environnement Windows Server. Il a fallu combiner les services web (Apache/PHP/MySQL via Wampserver) et les transferts sécurisés via FileZilla Server.

2. Déploiement de Wampserver

L'installation s'est effectuée sur une machine Windows Server. Les étapes clés :

  • Vérification préalable des packages Visual C++ avec l'outil Check_vredist pour s'assurer de la compatibilité.
  • Installation d'Apache, PHP, et MySQL.
  • Configuration du fichier httpd-vhosts.conf et de phpMyAdmin pour autoriser l'accès depuis le réseau local.

3. Configuration du Serveur FTP (FileZilla)

Afin de permettre l'envoi de fichiers sur les espaces web, le serveur FileZilla a été configuré avec précision :

  • Création de l'utilisateur de service dev_gsb avec un mot de passe fort.
  • Assignation du répertoire racine vers le dossier www d'Apache avec des droits restreints (Lecture, Écriture, Suppression limitées).
  • Configuration du mode passif (Passive Mode) en allouant la plage de ports 42000 à 42300 pour les transferts de données, et création des règles associées dans le pare-feu Windows.
